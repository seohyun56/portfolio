미디어 쿼리

미디어 쿼리를 활용하여 모바일, 태블릿, pc에 대응할 수 있는 반응형 홈페이지 구현

nav 배경색 변경

로고 색상 변경

Math.random 객체를 통해 배열 속 색상을 랜덤으로 호출하여 글자색 변경

project container

배열과 forEach를 사용하여 project 박스 클릭 시 location.href를 통해 프로젝트 페이지로 이동

skill container

::after를 사용하여 skill 제목에 밑줄 효과 구현

go up 버튼

addEventListener("scroll")을 통해 scrollY > 150일 때 opacity값 1로 변경

5. 문제 및 해결

로고 랜덤 글자색의 범위를 줄일 수는 없을까?

랜덤의 범위를 0 ~ 255로 줬더니 눈에 잘 보이지 않는 색까지 포함되는 문제점 발생

이를 해결하기 위해 원하는 색상을 정해 배열에 넣은 후 배열에 있는 색상을 랜덤으로 호출했더니 해결됨

프로젝트 페이지로 연결되는 div 박스 전체에 a를 연결할 수는 없을까?

배열 안에 프로젝트와 연결되는 링크를 넣어두고 이를 forEach와 location.href를 통해 배열 속 링크를 연결했더니 해결됨

skill 제목에 ::after를 적용했을 때 가로의 길이가 제목의 글자 길이와 같게 할 수 없을까?

::after를 통해 구현한 밑줄 효과의 가로 길이가 글자보다 길어지는 문제점 발생

skill의 제목을 감싸는 div에 align-items: flex-start을 적용하여 교차축 기준 시작점에 배치했더니 해결됨

pc버전에서 마우스 커서가 화면 밖으로 넘어가면 왜 가로스크롤이 생길까?

화면 밖으로 넘치면 보여지지 않게 하기 위해 body에 overflow-x: hidden을 줬더니 해결됨
