3. 페이지 구성

nav

nav 스크롤 이벤트

이전 scrollY가 현재 scrollY보다 작을 경우 nav style top의 값을 -(nav의 offsetHeight)px만큼 지정

이전 scrollY가 현재 scrollY보다 크거나 같을 경우 nav style top의 값을 0으로 지정

nav 배경색 변경

addEventListener("scroll")을 통해 scrollY > 0일 때 classList.add와 classList.remove를 이용하여 변경

header

swiper 이미지

swiper js를 사용하여 자동으로 넘어가는 swiper 이미지 구현

AOS 라이브러리

aos 라이브러리를 사용하여 페이지 로딩 시 fade-down 효과 구현

data-aos(효과), data-aos-easing(재생 속도), data-aos-duration(재생 시간) 사용

main

img animation

svg path와 css animation 기능을 통해 패턴 이미지에 움직이는 선 구현

background-position을 이용하여 계속 재생되는 메뉴 이미지 구현

AOS 라이브러리

aos 라이브러리를 사용하여 스크롤 시 fade, fade-up, fade-down, fade-left, fade-right 효과 구현

data-aos, data-aos-easing, data-aos-duration 사용

마우스 좌표

브라우저 기준 마우스 좌표인 clientX, clientY를 사용

마우스를 움직이면 circle 이미지들이 좌표에 따라 움직이는 효과 구현

footer

li hover시 밑줄 효과

::before를 통해 transform: scale(0);과 transform-origin: top-right; 적용

transition의 속성값을 transform으로 하여 hover시 scale(1), transform-origin: top-left; 적용

4. 문제 및 해결

nav의 메뉴 hover시 소메뉴와 배경이 같이 나오게 할 수 없을까?

대메뉴 ul인 .gnb와 형제관계인 gnbHover박스 생성

.gnb:hover시 소메뉴의 배경 역할을 하는 gnbHover박스가 나오게 했더니 해결됨

nav의 마지막 메뉴 hover시 div가 안 나오게 할 수 없을까?

addEventListener("mouseover")와 addEventListener("mouseout")사용

메뉴에 마우스를 올리면 div의 opacity값 0, 마우스가 떠나면 1이 되게 했더니 해결됨

넘치는 텍스트를 어떻게 구현해야 할까?

dalkommStory의 .storyTxt h3에 text-overflow: ellipsis를 사용하려 했으나 적용이 안 됨

시각적으로 똑같이 보이게 하기 위해 텍스트 수정

select의 화살표를 안 보이게 할 수 없을까?

select의 css에 apperance: none;을 적용했더니 해결됨
